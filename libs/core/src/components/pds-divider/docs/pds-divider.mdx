# Divider

A divider is used to create a clear visual separation between sections of content within a layout.

## Guidelines

### When to use
- To visually separate distinct sections of content for better readability.
- To create full-width or full-height boundaries within a section or page.

### When not to use
- When the separation between sections is already visually clear due to ample spacing.
- If the layout relies on other visual elements, such as cards or background shading, to define boundaries.
- When excessive dividers could create visual clutter or detract from content hierarchy.

### Accessibility
- Ensure sufficient contrast between the divider and the background to meet WCAG contrast requirements.
- Avoid using dividers as the sole method of separating content; support visual indicators with clear headings or labels.

## Examples

### Default

The default divider displays a horizontal line.

<DocCanvas
  mdxSource={{
    react: '<PdsDivider />',
    webComponent: '<pds-divider></pds-divider>'
}}>
  <div style={{width: '100%'}}>
    <pds-divider></pds-divider>
  </div>
</DocCanvas>

### Vertical

To display the divider vertically, set the `vertical` prop to `true`.

**Note**: Vertical dividers are not displayed on screen sizes smaller than 767px.

<DocCanvas
  mdxSource={{
    react: `
<div style={{ height: '150px' }}>
  <PdsDivider vertical="true" />
</div>
    `,
    webComponent: `
<div style={{ height: '150px' }}>
  <pds-divider vertical="true"></pds-divider>
</div>
    `
}}>
  <div style={{ height: '150px' }}>
    <pds-divider vertical="true"></pds-divider>
  </div>
</DocCanvas>

### Offsets

Both horizontal and vertical dividers can be offset to create a full-width or full-height bleed appearance.

Offset sizes follow the global spacing sizes: `xxs`: 4px, `xs`: 8px, `sm`: 12px, `md`: 24px, `lg`: 32px, `xl`: 48px, `xxl`: 64px.

<DocCanvas
  mdxSource={{
    react: '<PdsDivider offset="lg" />',
    webComponent: '<pds-divider offset="lg"></pds-divider>'
}}>
  <div style={{width: '100%'}}>
    <pds-divider offset="lg"></pds-divider>
  </div>
</DocCanvas>
