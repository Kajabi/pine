# Divider

Use the divider to separate sections of content. You can also use them to override auto-layout padding.

## Properties

<ArgsTable of="pds-divider" />

### Default

The default divider displays a horizontal line.

<Story story={stories.Default} />

<Canvas>
  <pds-divider />
</Canvas>

### Vertical

To display the divider vertically, the `vertical` prop can be set to `true`.

**Note**: Vertical dividers are not displayed on small screen sizes less than 767px.

<Story story={stories.Vertical} />

<Canvas>
  <div style={{ height: '150px' }}>
    <pds-divider vertical="true" />
  </div>
</Canvas>

### Offsets

Both horizontal and vertical dividers can be offset to create a full width/height bleed appearance.

Offset sizes follow the global spacing sizes: `xxs`:4px, `xs`:8px, `sm`:12px,`md`:24px, `lg`:32px, `xl`:48px, `xxl`:64px.

<Canvas>
  <pds-divider offset="lg" />
</Canvas>

Below you will see a vertical divider using a large offset.

<Canvas>
  <div style={{ height: '150px' }}>
    <pds-divider offset="lg" vertical="true" />
  </div>
</Canvas>
