# Switch

<p>
  Checkbox or radio component styled to appear as a toggle.
</p>

## Guidelines
### Usage

#### Checkbox or Radio
- If only one Switch is needed for users to enable/disable a value, use a `checkbox`
- Like radio buttons in forms, a `radio` Switch requires **at least two** related `radio` Switch inputs for users to 'toggle' between the different values
- `radio` Switches require a shared `name` along with an individual `component-id` for each Switch
