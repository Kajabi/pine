# Container

Container constrains its children to a semantic max-width, centering them within the available space. Use it to wrap form panels, settings pages, and full-page flows.

## Guidelines

### When to use

- Form panels and dialogs where content should be narrower than the viewport
- Settings pages and content areas that need a consistent reading width
- Page-level wrappers that cap the maximum site width

### Named sizes

| Size | Max-width | Common use |
|------|-----------|------------|
| `sm` | 340px | Narrow forms, confirmations |
| `md` | 520px | Standard forms, settings panels |
| `lg` | 700px | Wide panels, content pages |
| `xl` | 1064px | Page-level wrappers |
| `full` | 1440px | Max-width site container |

### Arbitrary sizes

Pass any valid CSS length string to `size` when the named tokens don't fit (e.g. `size="640px"` or `size="50rem"`). Named tokens are preferred for consistency.

## Examples

### Default

Without a `size`, the container grows to fill its parent with no max-width constraint.

  display="block"

    <pds-box background-color="secondary" padding="md">

### Not Centered

Set `centered="false"` to align the container to the start of its parent instead of centering it.

<DocCanvas
  display="block"
  mdxSource={{
    react: `<PdsContainer size="md" centered={false}><div>Left-aligned</div></PdsContainer>`,
    webComponent: `<pds-container size="md" centered="false"><div>Left-aligned</div></pds-container>`
  }}>
  <pds-container size="md" centered="false">
    <pds-box background-color="secondary" padding="md">
      <pds-text tag="p">Left-aligned — centered=false.</pds-text>
    </pds-box>
  </pds-container>
</DocCanvas>

### Semantic Tag

Use `tag` to render the inner container as a semantic HTML element.

<DocCanvas
  display="block"
  mdxSource={{
    react: `<PdsContainer size="lg" tag="main"><h1>Page title</h1></PdsContainer>`,
    webComponent: `<pds-container size="lg" tag="main"><h1>Page title</h1></pds-container>`
  }}>
  <pds-container size="lg" tag="main">
    <pds-box background-color="secondary" padding="md">
      <pds-text tag="h2">Page title inside tag="main"</pds-text>
    </pds-box>
  </pds-container>
</DocCanvas>
