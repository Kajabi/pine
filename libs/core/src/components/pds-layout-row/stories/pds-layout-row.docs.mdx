# Row
Base on 12 column flexbox-based grid system

## Properties

<ArgTypes of={stories} />

## Variants

### Default
By default boxes with rows will be equal width if no size is specified.

<Canvas mdxSource={`
<pds-layout-row>
  <pds-layout-box>Item 1</pds-layout-box>
  <pds-layout-box direction="column">
    <p>Content 1</p>
    <p>Content 2</p>
  </pds-layout-box>
  <pds-layout-box>Item 3</pds-layout-box>
</pds-layout-row>
`}>
  <pds-layout-row
    class="demo-row"
    component-id="${args.componentId}" 
    gap="12px"
    min-height="100px"
  >
    <pds-layout-box>Item 1</pds-layout-box>
    <pds-layout-box direction="column">
      <p>Content 1</p>
      <p>Content 2</p>
    </pds-layout-box>
    <pds-layout-box>Item 3</pds-layout-box>
  </pds-layout-row>
</Canvas>

### Wrapping
If the size values equal more than 12, the boxes will wrap to the next line.

<Canvas mdxSource={`
<pds-layout-row>
  <pds-layout-box bordered size-xs="4">size-xs 4</pds-layout-box>
  <pds-layout-box bordered size-xs="10">size-xs 10</pds-layout-box>
</pds-layout-row>
`}>
  <pds-layout-row>
    <pds-layout-box bordered size-sm="4">size-xs 4</pds-layout-box>
    <pds-layout-box bordered size-sm="10">size-xs 10</pds-layout-box>
  </pds-layout-row>
</Canvas>

### Layout Properties
`pds-layout-row` creates the horizontal rows that are used to contain a set of columns and can be view on the [Layouts](/docs/foundations-layout--docs) page.
