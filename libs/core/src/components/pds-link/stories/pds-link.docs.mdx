# Link

<p>
  Link is mainly used as navigational element and usually appear within or directly following a paragraph or sentence. The "plain link" is used for interactive user generated text
  like product, post, and offer titles.
</p>

## Properties

<ArgTypes of={stories} />

## Variants

### External

An `external` link is designed to redirect users to an external website or resource.

Notably, it's styled visually with an icon to indicate its external nature and opens a new browser tab on interaction.

<Canvas mdxSource={`
<pds-link href="#some-anchor" external="true"></pds-link>
`}>
  <pds-link href="#some-anchor" external="true"></pds-link>
</Canvas>

### Inline

The `inline` variant displays a standard inline hyperlink. Visually, it maintains the typical underlined appearance of a hyperlink.

<Canvas mdxSource={`
<pds-link href="https://www.google.com"></pds-link>
`}>
  <pds-link href="https://www.google.com"></pds-link>
</Canvas>

### Plain

The `plain` variant displays a standard plain hyperlink. Visually, it does not include an underline.

<Canvas mdxSource={`
<pds-link href="#some-anchor" variant="plain"></pds-link>
`}>
  <pds-link href="#some-anchor" variant="plain"></pds-link>
</Canvas>

### With Custom Text (uses slot)

Providing custom text within the open and closing tags allows the default link text to be replaced with custom content.

<Canvas mdxSource={`
<pds-link href="#some-anchor">Overrides default use of href</pds-link>
`}>
  <pds-link href="#some-anchor">Overrides default use of href</pds-link>
</Canvas>

### Without custom text in slot

When no custom text is provided, the link will use the `href` as the link text.

<Canvas mdxSource={`
<pds-link href="#some-anchor"></pds-link>
`}>
  <pds-link href="#some-anchor"></pds-link>
</Canvas>
