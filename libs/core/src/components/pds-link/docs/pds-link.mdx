# Link

A hyperlink is used as a navigational element to direct users to another location on the website or application.

## Guidelines

### Buttons vs Links

In many current web design patterns, buttons and links have evolved to look visually identical. However, for best practices, the fundamental differences between the two should be considered before using. Links are used to navigate the user to another page. Buttons are used to perform an action on the current page or submit a form.

### Default

The default `inline` link. Visually, it maintains the underlined appearance of a hyperlink.

<DocCanvas client:only
  mdxSource={{

}}>
  <pds-link href="https://www.google.com"></pds-link>

### Colors

The `color` prop can be used to change the color of the link. Available colors are `secondary`, `accent`, and `danger`.

<DocCanvas client:only
  mdxSource={{
    react: `<PdsLink href="https://www.google.com" color="secondary"></PdsLink>
    <PdsLink href="https://www.google.com" color="danger"></PdsLink>
    <PdsLink href="https://www.google.com" color="accent"></PdsLink>
    `,
    webComponent: `<pds-link href="https://www.google.com" color="secondary"></pds-link>
    <pds-link href="https://www.google.com" color="danger"></pds-link>
    <pds-link href="https://www.google.com" color="accent"></pds-link>`
}}>
  <div style={{ backgroundColor: '#333', padding: '4px' }}><pds-link href="https://www.google.com" color="secondary"></pds-link></div>
  <pds-link href="https://www.google.com" color="danger"></pds-link>
  <pds-link href="https://www.google.com" color="accent"></pds-link>
</DocCanvas>

### Plain

The `plain` variant displays a standard plain hyperlink without an underline.

<DocCanvas client:only
  mdxSource={{
    react: '<PdsLink href="#some-anchor" variant="plain"></PdsLink>',
    webComponent: '<pds-link href="#some-anchor" variant="plain"></pds-link>'
}}>
  <pds-link href="#some-anchor" variant="plain"></pds-link>
</DocCanvas>

### External

An `external` link is styled with an icon to indicate that it will redirect users to an external website or resource in a new tab.

<DocCanvas client:only
  mdxSource={{
    react: '<PdsLink href="#some-anchor" external="true"></PdsLink>',
    webComponent: '<pds-link href="#some-anchor" external="true"></pds-link>'
}}>
  <pds-link href="#some-anchor" external="true"></pds-link>
</DocCanvas>

### Font Size

The `font-size` prop can be used to adjust the size of the link text. `sm` is 12px, `md` is 14px, and `lg` is 16px. `md` is the default size.

<DocCanvas client:only
  mdxSource={{
    react: '<PdsLink href="#some-anchor" font-size="sm"></PdsLink>',
    webComponent: '<pds-link href="#some-anchor" font-size="sm"></pds-link>'
}}>
  <pds-link href="#some-anchor" font-size="sm"></pds-link>
</DocCanvas>

### Custom Slot Text

Providing text within the open and closing tags allows the default link text to be replaced with custom text.

<DocCanvas client:only
  mdxSource={{
    react: '<PdsLink href="#some-anchor">Overrides default use of href</PdsLink>',
    webComponent: '<pds-link href="#some-anchor">Overrides default use of href</pds-link>'
}}>
  <pds-link href="#some-anchor">Overrides default use of href</pds-link>
</DocCanvas>

### No Custom Slot Text

When no custom text is provided, the link will use the `href` as the link text.

<DocCanvas client:only
  mdxSource={{
    react: '<PdsLink href="#some-anchor"></PdsLink>',
    webComponent: '<pds-link href="#some-anchor"></pds-link>'
}}>
  <pds-link href="#some-anchor"></pds-link>
</DocCanvas>
