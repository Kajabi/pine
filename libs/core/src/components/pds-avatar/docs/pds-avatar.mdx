# Avatar

The avatar component is used to display a thumbnail representation of an admin, customer, or business.

## Guidelines

### When to use
- Use the `Avatar` to visually represent a user or entity, such as a profile picture, admin, or customer.
- Ideal for applications where identifying users or roles with a small image is needed.

### When not to use
- Avoid using for large images or media that require detailed visibility.
- Do not use for items that need a descriptive label or detailed information beyond a simple visual representation.

### Accessibility
- Always include an `alt` text description for profile images, ensuring users with visual impairments understand the context.
