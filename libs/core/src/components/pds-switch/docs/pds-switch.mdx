# Switch

A styled checkbox component that functions as a toggle.

## Guidelines

### When to use
- Use the `Switch` when you need a binary choice in a compact form, like enabling or disabling a feature.
- Ideal for toggling options in settings or preferences.

### When not to use
- Avoid using for actions that require multiple options or a more complex selection (e.g., radio buttons).
- Do not use for situations where the user's choice needs to be explicitly confirmed.

### Accessibility
- The component includes `aria-describedby` when an error message is present, ensuring that screen readers announce the error after the label.
