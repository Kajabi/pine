# Grid
The Grid component is a mobile-first layout component that allows for the creation of complex layouts. It is based on the CSS Grid specification and is intended to be used in conjunction with the Grid Item component.

## Sub Components

### Grid Item
The `pds-grid-col` component is the column component that is used to create the grid layout. It is intended to be used as a child of the `pds-grid` component.

Grid items will automatically wrap to the next line if there is not enough space horizontally for the grid item. Any `gap`s used will apply vertical spacing between the rows.

## Additional Resources


Start / End
Simiar to offset, use start to denote the beginning of a column. It is meant to be paired with end to denote the end of a column. A start with no end will be size to one column

End
Think of end when spanning to the end of the grid. It is meant to be paired with start to denote the end of a column. An end with no start will be size to one column
Setting an end only causes the grid item to be one column length, that may not be correct

Nesting
Nesting allows to place a grid inside of a grid. This is useful for creating more complex layouts.
