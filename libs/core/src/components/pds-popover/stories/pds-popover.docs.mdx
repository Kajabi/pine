# Popover

Popovers are containers that float on top of existing content used to display transient additional content, actions, or menus.

## Variants

### Trigger
Whether your trigger is a `pds-button` or a different component or HTML element, it will be placed in the default slot between the opening and closing tags.
There can only be one trigger for the Popover.

### Content
The popover content is placed in a container with the slot, `content`. This is where you can place any HTML content you want to display in the popover.
