# Popover

Popovers are containers that float on top of existing content used to display additional content.

## Properties

<ArgTypes of={stories} />

## Variants

### Trigger
The trigger can be a button, `pds-button`, another Pine component, or an HTML element. There is just one trigger for each Popover, making its use straightforward.
There can only be one trigger for the Popover.

#### Button trigger

<Canvas mdxSource={`
<pds-popover placement="bottom-right" html-content="true">
  <div slot="content">
    <p><strong>This is a popover</strong></p>
  </div>
  <pds-button variant="secondary">Click</pds-button>
</pds-popover>
`}>
  <div style={{ height: '100px' }}>
    <pds-popover placement="bottom-start" html-content="true">
      <div slot="content">
        <p><strong>This is a popover</strong></p>
      </div>
      <pds-button variant="secondary">Click</pds-button>
    </pds-popover>
  </div>
</Canvas>

### Placement
Using the `placement` prop you can adjust where your popover will be displayed. The position of the popover and the arrow can be configured. The default placement is `right`.
The `placement` can accept valus such as :`top-end` or `bottom-start` The first segment in the placement prop is the position of the popover. The second word is alignment of the popover in relation to the trigger. If no second word is provided, the popover will be centered on the trigger.

### Arrow
By default the arrow is hidden on popovers. It can be shown by enabling it when `has-arrow` is `true`.

<Canvas mdxSource={`
<pds-popover has-arrow="true" placement="bottom-start">
  <div slot="content">
    <p><strong>This is a popover</strong></p>
    <p><strong>This is a popover</strong></p>
  </div>
  <pds-avatar dropdown="true"></pds-avatar>
</pds-popover>
`}>
  <div style={{ height: '150px' }}>
    <pds-popover has-arrow="true" placement="bottom-start">
      <div slot="content">
        <p><strong>This is a popover</strong></p>
        <p><strong>This is a popover</strong></p>
      </div>
      <pds-button variant="secondary">Click</pds-button>
    </pds-popover>
  </div>
</Canvas>
