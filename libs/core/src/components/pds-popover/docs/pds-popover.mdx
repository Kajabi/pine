# Popover

Popovers display contextual information when users hover over or keyboard focus on an element. They typically contain more information than tooltips and can include interactive elements.

## Guidelines
### Usage
- Use popovers for complex information that might overwhelm the main content.
- Consider using a tooltip for simpler explanations or brief labels.

#### When not to use
- Don't use popovers for critical information that must be readily available.

## Variants
### Default
The trigger element is placed in the default slot and can be text, an icon, or another HTML element.

### Content
Popover content can be text or include HTML elements using a named content slot.

### Positioning
Using the placement prop, you can control where the popover appears relative to the trigger.

### Arrow
The popover includes an arrow by default, but you can hide it with the `has-arrow` prop.
