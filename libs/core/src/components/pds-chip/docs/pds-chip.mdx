# Chip

Chips are used to inform users about the status of an object or to associate properties with a parent. Commonly used in tabular data to indicate status.

## Guidelines

### When to use
- To represent compact labels, statuses, or categories.
- Ideal for conveying additional metadata in components like tables, cards, or filters.
- Useful for dismissible tags or quick user actions.

### When not to use
- Avoid using Chips for long, descriptive information.
- Do not use Chips as a primary action element in a UI.

### Accessibility

The `dropdown` variant and the icon within the `tag` variant of the chip component both render as a button and will inherit all accessibility attributes of a standard HTML button.

## Tag

If the `variant` property is set to `"tag"`, the chip will be rendered in a tag style, altering its appearance and behavior accordingly.

Tag is often used to represent a selected item or a removable element and includes a close button that can be clicked to perform an associated action.
