# Chip

Chips are used to inform users of the status of an object or to relate properties to a parent. Commonly used in tabular data to indicate status.

## Properties

## Tag

If the `variant` property is set to `"tag"`, the chip will be rendered in a tag style, altering its appearance and behavior accordingly.

Tag is often used to represent a selected item or a removable element and includes a close button that can be clicked to perform an associated action.

<DocCanvas client:only
  mdxSource={{

    <pdsChip label="Success" sentiment="success" variant="tag" />
