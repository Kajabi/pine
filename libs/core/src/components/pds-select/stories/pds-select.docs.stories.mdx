# Select

This is select description text

## Sub Components

### Select Option

A subcomponent of the select component. This component is used to define the options of the select component.

### Accessibility
This select is keyboard accessible following the Comboxbox and Listbox best practices from the W3C. The select is also screen reader accessible.
