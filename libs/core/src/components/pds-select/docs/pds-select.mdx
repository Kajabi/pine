# Select

<p>
  The select field, when combined with a label, helper message, and error text, offers a comprehensive control that provides a menu of options for user selection. This ensures
  clarity and guidance for the user, enhancing the overall user experience.
</p>

## Guidelines

### Usage

#### When to use

- When you want to provide a simple list of options for the user to choose from.
- **Example**: Use a select component to choose a state from a short list of U.S. regions.

#### When **NOT** to use

- If your list of options is greater than 10, consider using a custom dropdown component. (WIP)
- **Example**: Use a custom dropdown for selecting multiple filters or choosing a specific product from a comprehensive list. (WIP)

### Accessibility

- The select component comes with a built-in label through the `label` property, it is strongly encouraged that you include it.
- Please provide clear and descriptive error, and helper-message text.
- Selects can have a default option that is not selectable.
