# Input

<p>
  An input field coupled with a label and error text which allows for user input and feedback.
</p>

## Guidelines
### Usage

#### When to use
- To gather basic usage information
- You want a response that is one line of text

#### When not to use
- You want a response longer than one line, use a [Textarea](/docs/components-textarea--default)
- You want a response from a predefined list, use a select (TODO: create select and link here)


### Accessibility
- The input component comes with a built-in label through the `label` property, it is strongly encouraged that you include it
- Please provide clear and descriptive error, helper-message, and placeholder messages
- Do not use a `placeholder` to communicate the field label, use a `label` property
