# Sortable

Sortable is for creating containers of items that can be sorted via drag and drop functionality.

## Guidelines

### When to use
- Use the `sortable` component when you need to create an interface for reordering items through drag and drop.
- Ideal for situations where users need to arrange content dynamically.

### When not to use
- Avoid using for small lists or static content where drag-and-drop functionality is unnecessary.
- Not suitable for forms or content that does not require item reordering.

### Accessibility
- Ensure that drag-and-drop interactions are accessible by providing clear visual handles for reordering.

### Actions

The `enable-actions` property in the `pds-sortable-item` component determines whether the `sortable-item-actions` slot should be enabled.

The `sortable-item-actions` slot is provided to place content within the `pds-sortable-item__actions` element as children, but this slot is only rendered if the `enable-actions` property is set to `true`.
