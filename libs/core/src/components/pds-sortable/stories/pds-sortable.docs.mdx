# Sortable

Sortable is for creating containers of items that can be sorted via drag and drop functionality.

## Properties

<ArgTypes of={stories} />


## Sortable Items

The `pds-sortable-item` sub-component are the items that can be sorted within a sortable container `pds-sortable`.

<ArgTypes of={SortableItemStories} />

### Handle

When `handle` is set to `true` on a `pds-sortable-item` a handle icon will be displayed within an item.

  <pds-sortable-item handle>Item 2</pds-sortable-item>
  <pds-sortable-item handle>Item 3</pds-sortable-item>
</pds-sortable>
`}>
  <pds-sortable component-id="handle">
    <pds-sortable-item handle>Item 1</pds-sortable-item>
    <pds-sortable-item handle>Item 2</pds-sortable-item>
    <pds-sortable-item handle>Item 3</pds-sortable-item>
  </pds-sortable>
</Canvas>

#### Handle vs. Row `handle-type`
By default the full `row` is active for dragging/sorting. However, `handle-type` can be set to `handle` in order to allow only the handle to be used.

This should be used in combination with `handle` set to `true` on the `pds-sortable-item`.

<Canvas mdxSource={`
<pds-sortable component-id="handle-only" handle-type="handle">
  <pds-sortable-item handle>Item 1</pds-sortable-item>
  <pds-sortable-item handle>Item 2</pds-sortable-item>
  <pds-sortable-item handle>Item 3</pds-sortable-item>
</pds-sortable>
`}>
  <pds-sortable component-id="handle-only" handle-type="handle">
    <pds-sortable-item handle>Item 1</pds-sortable-item>
    <pds-sortable-item handle>Item 2</pds-sortable-item>
    <pds-sortable-item handle>Item 3</pds-sortable-item>
  </pds-sortable>
</Canvas>

### Actions

The `enable-actions` property in the `pds-sortable-item` component determines whether the `sortable-item-actions` slot should be enabled.

The `sortable-item-actions` slot is provided to place content within the `pds-sortable-item__actions` element as children, but this slot is only rendered if the `enable-actions` property is set to `true`.

<Canvas mdxSource={`
<pds-sortable component-id="actions">

    <div>Item 1</div>
    <div slot="sortable-item-actions">

    </div>
  </pds-sortable-item>

</pds-sortable>
