# Box

Box is our primitive layout component. It is intended to be used as a building block for more complex layouts. It allows for custom sizing, colors, padding, and alignment while still adhering to Pine tokens and brand guidelines.

## Variants

### `border`
By default Box has no border. Use `border` to add a border to the box. Use `borderColor` to change the color of the border. Use `borderRadius` to change the rounding of the box's corners.

### Spacing Properties Priority
The Box component implements a consistent pattern for spacing properties (padding, margin, gap) with different levels of specificity. Each spacing property follows this hierarchy:

1. General property (e.g., `padding`, `margin`, `gap`) - Sets spacing on all sides (lowest priority)
2. Block/Inline property (e.g., `paddingBlock`, `marginInline`, `gapRow`) - Sets spacing on top and bottom or left and right (medium priority)
3. Directional property (e.g., `marginLeft`) - Sets spacing on a specific side (highest priority)

When multiple spacing properties are set, the more specific properties take precedence. For example, with padding:

<DocCanvas client:only
  mdxSource={{
    react: `<PdsBox border="true" padding="md" paddingBlock="lg">
  padding="md" is overridden by paddingBlock="lg"
</PdsBox>
<PdsBox border="true" padding="md" paddingInline="lg">
  padding="md" is overridden by paddingInline="lg"
</PdsBox>`,
    webComponent: `<pds-box border="true" padding="md" padding-block="lg">
  padding="md" is overridden by padding-block="lg"
</pds-box>
<pds-box border="true" padding="md" padding-inline="lg">
  padding="md" is overridden by padding-inline="lg"
</pds-box>`
}}>
  <pds-box border="true" padding="md" padding-block="lg">
    padding="md" is overridden by padding-block="lg"
  </pds-box>
  <pds-box border="true" padding="md" padding-inline="lg">
    padding="md" is overridden by padding-inline="lg"
  </pds-box>
</DocCanvas>

This priority system allows for more granular control over spacing while maintaining a predictable behavior. When you need to set different spacing values for different sides, use the more specific properties (block and inline) to override the general property.

### Display
By default Box is `display: inline-flex`. Use `display` to change the display property `flex` if the box is intended to fill the entire row.

### Direction
By default Box is `flex-direction: row`. Use `direction` to change the direction property `column` if the box's contents are intended to stack vertically'.

<DocCanvas client:only
  mdxSource={{
    react: `<PdsBox border="true">

  <span>Item 2</span>
</PdsBox>

  <span>Item 1</span>
  <span>Item 2</span>
</PdsBox>`,
    webComponent: `<pds-box border="true">

  <span>Item 2</span>
</pds-box>
<pds-box border="true" direction="column">

  <span>Item 2</span>

### Layout Properties
`pds-box` not only allows for custom element properties, but it also allows for a flexible layout system. The layout system is based on the [CSS Flexbox specification](https://drafts.csswg.org/css-flexbox/) and can be view on the [Layouts](/docs/components-layout--docs) page
