# Box
Box is our primitive layout component. It is intended to be used as a building block for more complex layouts. It allows for custom sizing, colors, padding, and alignment while still adhering to Pine tokens and brand guidelines.

## Properties

<ArgTypes of={stories} />

## Variants

### Border
By default Box has no border. Use `border` to add a border to the box. Use `borderColor` to change the color of the border. Use `borderRadius` to change the rounding of the box's corners.

<Canvas mdxSource={`

### Padding

Use `paddingTop` to add padding to the top of the box. Use `paddingBottom` to add padding to the bottom of the box. Use `paddingLeft` to add padding to the left of the box. Use `paddingRight` to add padding to the right of the box.

### Display
By default Box is `display: block`. Use `display` to change the display property `flex` if the box is intended to fill the entire row. Use `inline-flex` if the box is intended to sit next to the previous or next box.

### Alignment
Adding either `flex` **`display`** property to the box will initiate alignment values. Use `align-items` to change the vertical alignment of the box. Use `justify-content` to change the horizontal alignment of the box.

    padding="sm"
  >
    Content 1
  </pds-box>
  <pds-box
    align-items="center"

    padding="sm"
  >
    Content 3

</pds-row>
`}>
