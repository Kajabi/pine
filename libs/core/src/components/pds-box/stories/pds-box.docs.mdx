# Box
Box is our primitive layout component. It is intended to be used as a building block for more complex layouts. It allows for custom sizing, colors, padding, and alignment while still adhering to Pine tokens and brand guidelines.

## Properties

<ArgTypes of={stories} />

## Variants

### Border
By default Box has no border. Use `border` to add a border to the box. Use `borderColor` to change the color of the border. Use `borderRadius` to change the rounding of the box's corners.

### Padding

Use `paddingTop` to add padding to the top of the box. Use `paddingBottom` to add padding to the bottom of the box. Use `paddingLeft` to add padding to the left of the box. Use `paddingRight` to add padding to the right of the box.

### Display
By default Box is `display: inline-flex`. Use `display` to change the display property `flex` if the box is intended to fill the entire row.

### Direction
By default Box is `flex-direction: row`. Use `direction` to change the direction property `column` if the box's contents are intended to stack vertically'.
<Canvas mdxSource={`
<pds-box
  bordered="true"
>

  <span>Item 2</span>
</pds-box>
<pds-box
  bordered="true"
  direction="column"
>
  <span>Item 1</span>
  <span>Item 2</span>
</pds-box>
`}>

    bordered="true"

    <span>Item 1</span>

### Layout Properties
`pds-box` not only allows for custom element properties, but it also allows for a flexible layout system. The layout system is based on the CSS Flexbox specificatio and can be view on the [Layouts](/docs/foundations-layout--docs) page
