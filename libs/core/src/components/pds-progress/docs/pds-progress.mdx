# Progress

The Progress component shows the progress of a task or process. It's great for helping users understand whatâ€™s happening and how much longer they might have to wait.

## Usage Guidelines

### When to use

- Use it to show the progress, like loading data or saving a file.
- Great for multi-step tasks, like creating a new account or filling out a form.
- Perfect for tasks that take a while, like uploading a large file.

### When not to use

- Avoid using for actions that complete quickly, like opening a modal or changing a setting.
- Not suitable for tasks that don't follow a linear path, like live updates.

## Accessibility

- Make sure to provide a clear and descriptive label for the progress indicator, like "Uploading file" or "Creating account".

### Percent

The percent of how much the `pds-progress` is filled. Should be a number ranging from 0 to 100.

<DocCanvas client:only mdxSource={{
  react: `<PdsProgress componentId="percent" label="percent" percent="25"></PdsProgress>`,
  webComponent: `<pds-progress component-id="percent" label="percent" percent="25"></pds-progress>`
}}>
  <pds-progress component-id="percent" label="percent" percent="25"></pds-progress>
</DocCanvas>

### Show Percent

Displays the `percent` value as text beside the progress.

<DocCanvas client:only mdxSource={{
  react: `<PdsProgress componentId="show-percent" label="Show percent" percent="50" showPercent={true}></PdsProgress>`,
  webComponent: `<pds-progress component-id="show-percent" label="Show percent" percent="50" show-percent></pds-progress>`
}}>
  <pds-progress component-id="show-percent" label="Show percent" percent="50" show-percent></pds-progress>
</DocCanvas>

### Animated

When `animated`, progress will animate from 0 to the `percent` value.

<DocCanvas client:only mdxSource={{
  react: `<PdsProgress componentId="animated" label="Animated " percent="75" animated={true}></PdsProgress>`,
  webComponent: `<pds-progress component-id="animated" label="Animated " percent="75" animated></pds-progress>`
}}>
  <pds-progress component-id="animated" label="Animated " percent="75" animated></pds-progress>

### Fill Color

Allows for `pds-progress` to be filled with a custom color. Accepts any valid color token or CSS color value.

<DocCanvas client:only mdxSource={{

`}}>
  <pds-progress component-id="fillcolor" label="Fill color" percent="95" fill-color="#86D5BC"></pds-progress>
</DocCanvas>
