# Tabs

The Tabs component is used to separate and display different panels of related content activated by tab buttons.

## Guidelines

### Accordions vs Tabs

Though similar, tabs and accordions both offer different user experiences to keep in mind. Consider using accordions for shorter, independent content and on devices with smaller viewports and limited space. Consider using tabs for longer, related content under a main topic.

### When to Use

- To display longer, related content under a main topic.

### When Not to Use

- If a longer label is desired. Consider using an accordion instead.

## Accessibility

For best practices in accessibility, be sure to provide a `tablist-label` and a unique `component-id`. The `tabslist-label` will provide context to screen readers about what kind of tabslist information is being presented.
