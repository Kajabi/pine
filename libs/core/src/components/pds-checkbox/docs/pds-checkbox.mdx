# Checkbox

Checkboxes provide users with selectable options like toggling a single setting or selecting multiple options from a list.

## Guidelines

### When to use
- Use checkboxes when users can select one or more options from a set.
- Ideal for situations where multiple selections can be made.

### When not to use
- Avoid using checkboxes for single-choice selections, such as in forms where only one option should be selected. In such cases, consider using radios or a switch.

### Accessibility
- Provide a clear, descriptive label for each checkbox to ensure that screen readers can identify the option effectively.

## Events

### pdsCheckboxChange
Used when you want to perform an action when the checkbox state changes. The returned structure is a `CheckboxChangeEventDetail` that contains `checked` and `value`.
