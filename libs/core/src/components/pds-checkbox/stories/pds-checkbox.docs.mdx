# Checkbox

Checkboxes provide users with selectable options like toggling a single setting or selecting multiple options from a list.

## Properties

<ArgTypes of={stories} />

### Default

<Canvas mdxSource={`
<pds-checkbox component-id="default" label="Label" />
`}>
<pds-checkbox component-id="default" label="Label" />
</Canvas>

### Checked

Checkboxes appear as checked when activated.

<Canvas mdxSource={`
<pds-checkbox component-id="checked" label="Label" checked />
`}>
  <pds-checkbox component-id="checked" label="Label" checked />
</Canvas>

### Indeterminate

Component `indeterminate` prop is for visual styling only.

Only JavaScript can set the objects `indeterminate` property. See [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/checkbox#indeterminate_state_checkboxes)

<Canvas mdxSource={`
<pds-checkbox component-id="indeterminate" label="Label" indeterminate checked />
`}>
  <pds-checkbox component-id="indeterminate" label="Label" indeterminate checked />
</Canvas>

### Disabled

Disabled visually indicates and functionally prevents interaction with the checkbox.

<Canvas mdxSource={`
<pds-checkbox component-id="disabled" label="Label" disabled />
`}>
  <pds-checkbox component-id="disabled" label="Label" disabled />
</Canvas>

### Message

Message is a string of text that displays below the label text.

<Canvas mdxSource={`
<pds-checkbox component-id="message1" label="Label" helper-message="This is short message text." />
`}>
  <pds-checkbox component-id="message1" label="Label" helper-message="This is short message text." />
</Canvas>

### Invalid

Invalid displays the checkbox in invalid state such as when required but is not checked.

<Canvas mdxSource={`
<pds-checkbox component-id="invalid" label="Label" invalid />
`}>
  <pds-checkbox component-id="invalid" label="Label" invalid />
</Canvas>
