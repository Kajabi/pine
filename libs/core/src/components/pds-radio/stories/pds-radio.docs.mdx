# Radio

Radio components provide users a way to select only one option from a list of two or more options.

## Properties

<ArgTypes of={stories} />

### Default

<Canvas mdxSource={`
  <pds-radio component-id="default" label="Label" />
`}>
  <pds-radio component-id="default" label="Label" />
</Canvas>

### Checked

Radios appear as checked when activated.

<Canvas mdxSource={`
  <pds-radio component-id="checked" label="Label" checked />
`}>
  <pds-radio component-id="checked" label="Label" checked />
</Canvas>

### Disabled

Disabled visually indicates and functionally prevents interaction with the radio.

<Canvas mdxSource={`
  <pds-radio component-id="disabled" label="Label" disabled />
`}>
  <pds-radio component-id="disabled" label="Label" disabled />
</Canvas>

### Message

Message is a string of text that displays below the label text.

<Canvas mdxSource={`
  <pds-radio component-id="message1" label="Label" helper-message="This is short message text." />
`}>
  <pds-radio component-id="message1" label="Label" helper-message="This is short message text." />
</Canvas>

### Invalid

Invalid displays the radio in invalid state such as when required but is not checked.

An optional error message can be displayed for additional instructions, displayed independently from the helper message.

<Canvas mdxSource={`
  <pds-radio component-id="invalid" label="Label" invalid error-message="This is a short error message" />
`}>
  <pds-radio component-id="invalid" label="Label" invalid error-message="This is a short error message" />
</Canvas>

### Group

A radio group is defined by assigning each radio in the group the same `name`.

Once a radio group is created, only one radio at a time can be selected.

<Canvas mdxSource={`
<div>
  <pds-radio component-id="default1" label="Label" name="radio" checked />
  <pds-radio component-id="default2" label="Label" name="radio" />
  <pds-radio component-id="default3" label="Label" name="radio" />
</div>
`}>
  <>
    <pds-radio component-id="default1" label="Label" name="radio" checked />
    <pds-radio component-id="default2" label="Label" name="radio" />
    <pds-radio component-id="default3" label="Label" name="radio" />
  </>
</Canvas>
