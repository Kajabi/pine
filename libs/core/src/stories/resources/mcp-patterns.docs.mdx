# Admin composition patterns

Verified page-level Pine recipes for Kajabi admin screens. Use these when assembling settings pages, list indexes, filters, confirmations, and empty/loading states — not when looking up individual component props.

> **Source of truth:** `libs/core/src/stories/resources/mcp-patterns.json` is consumed by Storybook (this page) and by [pine-mcp](https://github.com/Kajabi/pine-mcp) (`list_pine_patterns` / `get_pine_pattern`). Edit the JSON in Pine; do not duplicate recipes in pine-mcp.
